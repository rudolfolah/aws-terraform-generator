# aws-terraform-generator
Generate Terraform configurations for existing AWS infrastructure resources

Based on [aws2tf](https://github.com/aws-samples/aws2tf)

## Usage

```
AWS_PROFILE=example node index.js
AWS_PROFILE=example AWS_SDK_LOAD_CONFIG=1 node index.js
AWS_ACCESS_KEY_ID=example AWS_SECRET_ACCESS_KEY=example node index.js
```
